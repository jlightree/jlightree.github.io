# 주제: [[HTML]]
# `<!doctype html>
- 이 웹 페이지가 HTML로 만들어졌음을 나타내는 관용적인 태그
# `<html></html>
- `<head>`와 `<body>`를 감싸는 태그
# `<head></head>`
- 본문을 설명하는 태그

## `<title></title>`
- 웹 브라우저의 탭에 태그 안의 내용을 보여준다.
- 검색엔진이 웹 페이지를 분석할 때 가장 중요하게 여기는 태그다.

## `<meta>`
- 본문을 설명하는 태그 중 하나이다
- `<meta chaset="utf-8">`
	- 코드가 utf-8로 쓰였음을 나타낸다.

# `<body></body>`
- 본문이 들어가는 태그
