# 주제: [[HTML]]
- 태그에는 부모, 자식 태그의 개념이 있다.
```HTML
<p>
	<a href="https://opentutorials.org">opentutorials</a>
</p>
```
- 위 상황에선 p 태그가 a 태그의 부모 / a 태그가 p 태그의 자식이다.
- 대부분의 태그는 부모 자식 관계가 변할 수 있는 반면 부모 자식 관계가 고정된 태그들이 있다.
```HTML
<ul>
	<li>1.HTML</li>
	<li>2.CSS</li>
	<li>3.JavaScript</li>
<ul>
<ol>
	<li>HTML</li>
	<li>CSS</li>
	<li>JavaScript</li>
<ol>
```
- 1.HTML
- 2.CSS
- 3.JavaScript

1. HTML
2. CSS
3. JavaScript
